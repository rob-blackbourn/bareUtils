bareutils.header

module bareutils.header ¶

Summary¶

Header utilities

Description¶

A collection of functions to extract headers from the ASGI scope.

function bareutils.accept ¶

Summary¶

Returns the accept header if it exists.

Description¶

Where quality is not given it defaults to 1.0.

>>> accept([(b'accept', b'text/html, application/xhtml+xml, application/xml;q=0.9, /;q=0.8')])
{b'text/html': 1.0, b'application/xhtml+xml': 1.0, b'application/xml': 0.9, b'/': 0.8}
bareutils.accept(
headers: Iterable[Tuple[bytes, bytes]],
*,
add_wildcard: bool,
default: Optional[Mapping[bytes, float]]
) -> Optional[Mapping[bytes, Tuple[bytes, Any]]]

Parameters¶

headers: Iterable[Tuple[bytes, bytes]]

The headers

add_wildcard: bool (optional)

If true add the implicit wildcard '*'. Defaults to False.

default: Optional[Mapping[bytes, float]] (optional)

An optional default. Defaults to None.

Returns¶

Optional[Mapping[bytes, Tuple[bytes, Any]]]: A dictionary where the key is media type and the value is quality.
